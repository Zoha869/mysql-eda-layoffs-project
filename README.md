MySQL Exploratory Data Analysis – Layoffs Dataset
📌 Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) using MySQL on a global layoffs dataset.

The objective is to explore patterns in layoffs across companies, industries, countries, and time periods, and extract meaningful business insights using SQL queries.

🎯 Objectives
Analyze global layoff trends
Identify top companies affected by layoffs
Understand industry-wise and country-wise impact
Study yearly and monthly trends
Apply advanced SQL techniques for insights
🛠️ Tools & Technologies
MySQL
SQL
Window Functions
CTEs (Common Table Expressions)
Aggregation Functions
📂 Dataset

The dataset contains information about company layoffs including:

Company name
Industry
Country
Date
Number of layoffs
Funding raised
Stage of company
📊 Key Analysis Performed

✔ Maximum layoffs and percentage layoffs
✔ Company-wise total layoffs
✔ Industry-wise layoffs
✔ Country-wise layoffs
✔ Yearly layoff trends
✔ Monthly rolling totals
✔ Ranking of companies per year
✔ Top companies with highest layoffs

📈 Advanced SQL Concepts Used
GROUP BY aggregation
ORDER BY sorting
CTE (Common Table Expressions)
Window Functions (SUM OVER, DENSE_RANK)
Date extraction functions
Rolling totals calculation
🔍 Key Insights
Some companies had 100% workforce reduction
Tech industry experienced the highest layoffs
USA had the highest number of layoffs
Layoffs peaked in specific years globally
